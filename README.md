# NFT Generator

Generates NFT from parts.

## Env

- python 3.9
- `pip install Pillow blend_modes numpy`

## Data format

```csv
category,item,count,filepath,type,value,mark,filter,mode
Background,Aidan,24,element/Background/Aidan.png,Background,Aidan,,,
```

## Run

`python main.py`
